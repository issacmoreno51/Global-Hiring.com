# GLOBAL HIRING - Portal de Empleo Internacional

Sitio web para la agencia de empleo internacional GLOBAL HIRING, especializada en conectar talento con oportunidades laborales en el extranjero.

## Características

- Diseño responsivo
- Sección de países con oportunidades laborales
- Información sobre la empresa
- Formulario de contacto
- Integración con WhatsApp

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Maps API

## Cómo visualizar el proyecto

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. O visita la página desplegada en GitHub Pages

## Estructura del proyecto

- `index.html` - Página principal
- `style.css` - Estilos CSS
- `imagenes/` - Carpeta con todas las imágenes
- `README.md` - Documentación
